How to play the game:
*open the webpage
*select the levels and the number of guesses to set the difficulty 
*start guessing and refer to the clue after each guess you make;

Use of AI:
*I used claude and chatgpt free versions for basic troubleshooting and correcting bugs;
*Finding and correcting bugs in my script.js file (which were basically some syntax errors);
